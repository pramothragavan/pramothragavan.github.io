Interactive Open Quantum Walk

Below is a simulation of Machida’s model for the open quantum walk on $\mathbb{Z}$, for $N=50$ steps.

In the language of TOMs and quantum Markov chains, the initial vector state $S=(S_i)_{i\in\mathbb{Z}}$ is given by

\[ \ S_i = \begin{cases} \ket{L}\bra{L}+(1-p)\ket{R}\bra{R} & \text{if } i = 0, \\ 0 & \text{otherwise}, \end{cases} \] for some $p\in[0,1]$. Equivalently, in Machida’s original language, the walker’s initial state is given by the $\rho=\ket{0}\otimes(p\ket{0}\bra{0}+(1-p)\ket{1}\bra{1})\in\mathcal{H}_\text{pos}\otimes\mathcal{H}_d$, with the same constraints on $p$.

The simulation will take a few seconds to update as you vary parameters.
